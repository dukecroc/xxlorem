XXLOREM.js
一用就憤怒的中文假字生產器

這是什麼？

XXLOREM.js 是一款針對 前端設計師 打造的 中文假字生產器。

假字是什麼？能吃嗎？

假字是讓設計師用來填充版面的內容物，作為提案和設計過程中的暫時內容，而非讓大眾閱讀。

語料的來源是？

XXLOREM.js 的語料使用 靠北設計師 的匿名靠北內容。我們從中篩選了一些內容，並以程式隨機打亂後重新組合，作為假字使用。

有開源嗎？

你可以到 Github fork 這支 Plugin，歡迎任意複製、更改。

使用方法

XXLOREM.js 是以 jQuery 為基礎編寫，使用時需先引用 jQuery
請在你的網頁加上以下語法：

<script src="//code.jquery.com/jquery-1.11.3.min.js"></script>
<script src="//xxlorem.unfoldgroup.com/plugin/xxlorem-1.0.js"></script>

XXLOREM.js 提供了兩種方法，讓你可以快速將假字套用到頁面中：

方法一：以 jQuery 套用假字於特定物件上
$('.example').xxlorem();
$('.example').xxlorem(3);
方法二：直接在想要填入假字的物件加上 class="xx"
<p class="xx"></p>
<p class="xx(3)"></p>

參數設置

前述的兩種使用方法，都可以藉由設置不同的參數，來產生不同長度和類型的假字。

採用 方法一 時，您可以用 $(elment).xxlorem(params) 方式來設置參數；在 方法二 中，則可以用增加括弧和數值的方式 <p class="xx(params)"></p> 來設置參數。

XXLOREM.js 提供了多種類型的假字如下表：

類型 參數語法 簡寫 說明
文字 word , words w 產生一個或多個文字
句子 sentence , sentences s 產生一個或多個句子
段落 paragraph , paragraphs p 產生一個或多個段落文字
姓名 name n 產生一個假姓名
姓 last name ln 產生一個假姓氏
名 first name fn 產生一個假名字
姓名(拼音) name pinyin np 產生一個拼音假姓名
姓(拼音) last name pinyin lnp 產生一個拼音假姓氏
名(拼音) first name pinyin fnp 產生一個拼音假名字
Email email , mail e , m 產生一個假 email

在一般的狀態下，您只需使用到一個參數即可。

您可以僅使用一個參數，就完成設置假字的類型，或假句子的長度。此外，您也可以用縮寫的方式，同時設置類型和長度，例如 xx(3p) 即代表生成三個段落的文字內容。

在設定非數值參數時，若您使用 jQuery 套用假字，參數需以單引號或雙引號括住；若是直接寫在 HTML 標籤中，則不需要使用引號，例如：

$(elment).xxlorem('name');
$(elment).xxlorem(5);
<span class="xx(email)"></span>
<span class="xx(last name)"></span>
<span class="xx(3p)"></span>

當你在設定多個文字、句子、段落時，則可以使用到兩個參數，這兩個參數分別為 假字類型 和 產生長度。請以半形逗號隔開不同參數，參數的順序將不影響結果；若直接寫在 HTML 標籤時，逗號前後不能含有空白：

$(elment).xxlorem('sentences', 10);
$(elment).xxlorem(5, 'paragraphs');
<span class="xx(words,19)"></span>
<p class="xx(3,p)"></p>

在產生段落 (paragraphs) 時，你將可彈性的使用第三個參數。

第三個參數為設置段落時，允許改變換行符號，預設的換行符號為 \n\n，您可以替換為其它的符號，例如 <br><br> 或 <hr>：

$(elment).xxlorem(3, 'paragraphs', '<br><br>');
<p class="xx(6,p,<hr>)"></p>

雖然如此，但我們建議以縮寫、僅使用兩個參數的方式，來設置段落：

$(elment).xxlorem('3p', '<br><br>');
<p class="xx(6p,<hr>)"></p>

使用範例

1 個字：

Code <p class="xx(word)"></p>

你也可以這樣寫：xx(w)、xx(1w)、xx(1 word)

我需要 19 個字：

Code <p class="xx(19 words)"></p>

你也可以這樣寫：xx(19w)、xx(words,19)、xx(19_word)、xx(19-words)

麻煩隨便來個句子：

Code <p class="xx"></p>

你也可以這樣寫：xx(sentence)、xx(1)、xx(s)、xx(1s)

我需要大概 3 句文字：

Code <p class="xx(3s)"></p>

你也可以這樣寫：xx(sentences,3)、xx(3,s)、xx(3-sentences)

這邊來一個段落：

Code <p class="xx(p)"></p>

你也可以這樣寫：xx(paragraph)、xx(1p)

這篇文章需要 3 段，並把換行符號改為<br><br>：

Code <p class="xx(3p,<br><br>)"></p>

你也可以這樣寫：xx(3,p,<br><br>)、xx(paragraphs,3,<br><br>)

來個人名：

Code <p class="xx(name)"></p>

你也可以這樣寫：xx(n)

來個拼音的假姓名：

Code <p class="xx(name pinyin)"></p>

你也可以這樣寫：xx(np)、xx(name-pinyin)

來個 Email

Code <p class="xx(email)"></p>

你也可以這樣寫：xx(e)、xx(mail)、xx(m)
